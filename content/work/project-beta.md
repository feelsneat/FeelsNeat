As part of our commitment to building simple, practical utilities, we built an interactive layout sandbox. 

This tool helps designers and developers:
- Visualize vertical grid rhythms based on typographic line heights.
- Drag and adjust column spacing with instant CSS output.
- Export clean, zero-dependency Tailwind v4 styles.
